### 実習：GitHubのIssueを作成する

次の手順に従って、クラスのリポジトリでIssueを作成します。

1. *Issues*タブをクリック
1. *New Issue*をクリック
1. タイトルに次のとおりに入力　`YOUR-USERNAME Workflow`
1. Issueの本文に、次のテキストを含める

```md
YOUR-USERNAME はキャプションを変更し、ファイルを更新します。

- [ ] ブランチの作成
- [ ] ファイルの変更
- [ ] 変更のコミット
- [ ] Pull Requestの作成
- [ ] レビューのリクエスト
- [ ] 変更を追加
- [ ] 承認を受ける
- [ ] Pull Requestをマージ
```

1. *Submit new issue* ボタンをクリックし、issueを開きます。
